Welcome to the official page of the Mod the Gungeon project!

Mod the Gungeon is a set of tools to enable an easy start into Enter the Gungeon modding.

MtG's flagship utility is the Semi Mod Loader, which you can download below.

Semi is an open-source mod loader for Enter the Gungeon.

Please note that Semi doesn't add any end user features (such as items or enemies) by itself, as its sole purpose is to load mods and provide them easy ways to add and modify content.

Below are a couple of things that Semi allows you to do:

CONTENT!

Semi gives you full control over your custom content, while handling all the complicated boilerplate for you.

Test Gun Mk2 - Reimplemented in Semi

MODDING API

Semi aims to make creating custom content as easy as possible through it's Application Programming Interface.

Everything the API supports is based on named IDs, which removes risk of conflicts between mods and allows better integration.

SPRITING ON STEROIDS

Semi allows you to easily create sprites and entire animations with its custom file format.

Custom animation demo.

AUDIO

For the first time in Gungeon modding history - custom audio.

Supports streaming music and preloading sounds.

Install Semi

To install mods, you first need to install the Semi mod loader. To accomplish that, you need to use the Semi Graphical Installer.

Note - the Installer is not the Mod Loader, and the Mod Loader is not the Installer. This is often confused by people reporting issues. The installer merely installs the latest version of the Semi Mod Loader, and the installer version is not the same thing as the Semi version.

Select your operating system:
  • Requires .NET Framework 4.5+ on Windows.
  • SmartScreen might block the installer and call it a threat, risk or something else. SmartScreen is a ridiculous Microsoft scam program requiring a ridiculous amount of money for a "certificate" so that users don't receive the intentionally scary message. Ignore it or scan the installer with VirusTotal if you don't trust it.
  • The download is an .exe on all platforms - use Mono on Linux and Mac to run it. DO NOT USE WINE.

You have to select your OS to download the installer.
