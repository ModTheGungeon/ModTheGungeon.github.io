void SGUI.SGUIIMBackend.Dispose ( )

Releases all resource used by the SGUI.ISGUIBackend object.

Call Dispose() when you are finished using the SGUI.ISGUIBackend. The Dispose() method leaves the SGUI.ISGUIBackend in an unusable state. After calling Dispose(), you must release all references to the SGUI.ISGUIBackend so the garbage collector can reclaim the memory that the SGUI.ISGUIBackend was occupying.
