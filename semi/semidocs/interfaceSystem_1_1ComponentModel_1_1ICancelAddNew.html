Detailed Description

Interface implemented by a list that allows the addition of a new item to be either cancelled or committed.

Note: In some scenarios, specifically Windows Forms complex data binding, the list may recieve CancelNew or EndNew calls for items other than the new item. These calls should be ignored, ie. the new item should only be cancelled or committed when that item's index is specified.
