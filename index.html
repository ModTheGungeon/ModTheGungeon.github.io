Welcome to Mod the Gungeon!

Mod the Gungeon is an open-source modding ecosystem for "Enter the Gungeon". It consists of:

  • ETGMod, the Base backend for mods
  • ETGMod.Installer, the backend installer
  • ExampleAPI & ExampleMod, an example of a backend and a mod using that backend
  • InfKeysMod, a Base mod for infinite keys, can be used as an example of how to make mods for ETGMod.

It not only is a mod loader, but also a mod itself. Look at all the features we've got right now!

Download ETGMod.Installer

FOR Gungeon 1.0.10+

The Mod the Gungeon installer is how you install and update ETGMod. Either select the EtG.exe or - if you've got Steam - let the installer find it for you. Then select the backends and hit the install button! If you ever want to uninstall, just hit the red X next to the install button.

The version here is ETGMod.Installer.Citrus. The open-source section explains the various versions.

Open-Source on GitHub

Mod the Gungeon is an open-source project, by the community, for the community. Take a look at the source on GitHub, throw issues in our direction or even fork and send pull requests! Under the hood, the installer uses MonoMod, which is powered by Mono.Cecil. Both are also open-source!
