Welcome to Mod the Gungeon RELOADED !

Mod the Gungeon Reloaded is an upcoming rewritten version of Mod the Gungeon. Right now, it is heavily work in progress and is not ready for end user use yet.

Mod the Gungeon is an open-source modding platform for Enter the Gungeon. It consists of:

  • MTG Core - responsible for injecting all of these components
  • MTG Base - mod loader, APIs, GUI and console
  • TexMod - support for patching textures and animations
  • Installer.Headless, the installer

The main purpose of Mod the Gungeon is to be a mod loader, therefore it doesn't add any end user features (such as items or enemies) by itself.

Below is a list of what we have right now (note: some of this is slightly outdated):

CONSOLE

Test synergies, various scenarios, debug your mods, add your own commands - the possibilities are endless.

LUA MODDING

Write your mods in Lua, an extremely simple and small programming language with a clean and simple API.

Note: Mod the Gungeon mods used to be written in C#, this is no longer the case and C# is no longer supported for mods. The Lua API has also changed significantly, which is not reflected in the video.

TEXMOD

TexMod is a very powerful texture patching system that replaces the old basic sprite replacement method. It not only allows you to patch textures, but also patch and modify the flow of entire animations.

Install Mod the Gungeon Reloaded

The Mod the Gungeon Installer is how you install and update Mod the Gungeon. There is currently no available graphical installer for Reloaded.

Simply unpack the .zip, then run MTGInstaller install ModTheGungeon to install Reloaded.

NOTE: This is an installer, not the actual Mod the Gungeon. It simply downloads the chosen version of Mod the Gungeon and then installs it, but it is not Mod the Gungeon itself.

  • Requires .NET Framework 4.5+ on Windows and Mono on other platforms.
  • The download is an .exe on all platforms - use Mono on Linux and Mac to run it. DO NOT use Wine.
  • Don't worry if SmartScreen blocks it - it uses a stupid algorithm that just detects that the installer is modifying the game.

Open-Source on GitHub

Mod the Gungeon is an open-source project, by the community, for the community. Take a look at the source on GitHub, throw issues in our direction or even fork and send pull requests! Under the hood, the installer uses MonoMod, which is powered by Mono.Cecil. Both are also open-source!
